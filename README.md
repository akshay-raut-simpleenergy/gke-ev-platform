# GKE EV Platform

This repository contains the Kubernetes manifests and supporting documentation for deploying an EV telemetry platform on Google Kubernetes Engine (GKE).

## Components

- EMQX
- Apache Kafka
- Apache Flink
- Apache Pinot
- Apache Spark

## Repository Structure

- infrastructure/
- applications/
- scripts/
- docs/
- diagrams/

## Deployment Workflow

1. Update Kubernetes manifests.
2. Commit changes.
3. Push to GitHub.
4. Deploy to GKE using kubectl.
5. Verify the deployment.
